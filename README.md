# 📊 Calculadora de Notas

## 📌 Objetivo
Este projeto tem como objetivo calcular a média de um aluno e determinar sua situação final.

---

## 🧠 Como funciona
O usuário informa:

- Nome do aluno  
- Primeira nota  
- Segunda nota  

O sistema calcula:

- 📊 Média final  
- 📚 Situação do aluno (Aprovado, Recuperação ou Reprovado)  

---

## 🧮 Fórmula utilizada
Média:
(nota1 + nota2) / 2

Regras:

- Média ≥ 7 → Aprovado  
- Média ≥ 5 → Recuperação  
- Média < 5 → Reprovado
